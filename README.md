# sonomama-kaku

デスクトップ上に直接描画できるスクリーンオーバーレイツール

## インストール

[GitHub Releases](https://github.com/Andyyyy64/sonomama-kaku/releases) から最新版をダウンロードしてください。

- **Windows**: `.exe` ファイルを実行
- **macOS**: `.dmg` を開いてアプリケーションフォルダにドラッグ

## 使い方

1. アプリを起動するとシステムトレイにアイコンが表示されます
2. トレイアイコンをクリック、または `Ctrl+Shift+A` でオーバーレイを起動
3. 画面上に自由に描画できます
4. `Esc` キーでオーバーレイを終了
